Complete Design Summary

Trello-style /boards page for Planora

Page Structure

Top Navbar
  • Planora logo (left)
  • User avatar dropdown (right)
  • Dropdown contains: Profile, Create Workspace, Sign out
Sidebar
  • Boards button (top) → overview
  • Workspaces section (collapsible)
  • Each workspace has Boards link
Main Content
  • Overview: all workspaces + boards
  • Workspace view: single workspace boards
  • Board cards link to /boards/[id]

Views & States

Trigger Main Content Shows
Top "Boards" clicked All workspaces with their boards (overview)
Workspace → Boards clicked Only that workspace's boards
No workspaces exist Centered CTA: "Create your first workspace"
Board card clicked Navigate to /boards/[boardId] (kanban)

Create Workspace Flow

  • Modal dialog triggered from user dropdown OR empty state CTA
  • Same modal component for both entry points
  • After creation: new workspace appears in sidebar, auto-selected
